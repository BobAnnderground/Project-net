## Радиусы и эффекты

- `--border-radius/8` → 8px (root frame, кнопки Start/Enable full mode)
- `--border-radius/12` → 12px (hero-карточка, ServiceIcon, чип "+12")
- Аватар User — `rounded-[900px]` (полный круг, не токенизировано как отдельная переменная, просто большое число)
- `inner-shadow` эффект: `INNER_SHADOW`, цвет `Colors/White/24` (`rgba(255,255,255,0.24)`), offset `(0,0)`, radius `1`, spread `0` → на чипе "+12": `inset 0px 0px 1px 0px rgba(255,255,255,0.24)`
- `inner-button` эффект: `INNER_SHADOW`, цвет `Effects/Button-shadow` (`rgba(255,255,255,0.24)`), offset `(-2,4)`, radius `12`, spread `0` → на кнопке "Start": `inset -2px 4px 12px 0px rgba(255,255,255,0.24)`
- `icon-blur` эффект: `FOREGROUND_BLUR`, radius = `effects/icon-blur` = **4** — применяется к размытым цветным пятнам-подсветкам (5 штук, каждое ≈330×428px, замаскировано через общий alpha-mask `maste_canvas_1_key_1`, позиционированы по нижнему краю hero-карточки). Именно они дают декоративное свечение (фиолетовое/жёлтое/синее/белое), видимое на скриншоте.
